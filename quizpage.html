Q1. Which web browser was developed by Tim Berners-Lee in 1989?





Correct answer: World Wide Web browser


Q2. What significant feature did Mosaic introduce as a web browser?





Correct answer: Graphical display of multimedia


Q3. What was a significant reason for the discontinuation of Internet Explorer?





Correct answer: It faced heavy competition from other browsers.


Q4. Which of the following browsers were mentioned as having been discontinued? (Select all that apply)





Correct answer: Netscape Navigator, Internet Explorer


Q5. The first web browser, created by Tim Berners-Lee in 1989, was part of the ________ project at CERN. (Fill in the blank with lower case letters only)


Correct answer: World Wide Web
